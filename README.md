Excel Interactive Dashboard — Data Analytics Project
This is a hands-on data analytics project I built to practice Excel skills in a real-world scenario. The project covers the full data workflow — from raw, messy data to an interactive dashboard.

What I Did
Cleaned raw data (handled missing values, removed duplicates, standardized formats)
Processed the data using Excel functions and pivot tables
Analyzed the data to identify trends and business insights
Built an interactive dashboard with slicers and dynamic charts
Dashboard Features
Slicer-based filtering — selecting a value (e.g., a store or category) updates all charts instantly
Visual summaries of key metrics
Clean, professional layout suitable for reporting
Tools Used
Microsoft Excel (Pivot Tables, Slicers, Charts, Formulas)
Data cleaning and preparation techniques
Skills Gained
Data cleaning and transformation
Exploratory data analysis
Dashboard design and interactivity
Translating raw data into actionable insights
